Full Stack Development - Lab Manual

How to use:
  • Static Files: Click "View" to open directly (Practical 1, 2, 8)
  • React/Express Apps:
    1. Click "Show Command" to see the terminal command
    2. Click "Copy" and paste in PowerShell/Terminal
    3. Wait for server to start, then click "View"
  • Alternative: Double-click the .bat files in File Explorer to run them directly
  • Practical 17: Make sure MongoDB is running first
